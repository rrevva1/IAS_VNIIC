# Документация для написания ВКР

Навигатор по материалам проекта, полезным для выпускной квалификационной работы (ВКР).

## Два контура использования

1. **Документация на ИАС** — описание системы, требования, архитектура, модель данных (для разделов ВКР «Объект и предмет», «Требования», «Проектирование»).
2. **Методика и реализация** — правила формирования ТЗ, план реализации, отчёты о разработке (для разделов «Методы», «Реализация», «Внедрение»).
