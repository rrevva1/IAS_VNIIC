# ИАС УТС — информационно-аналитическая система учета технических средств

Репозиторий проекта ИАС учета технических средств предприятия (Help Desk, учет активов, справочники, аудит).

- **Приложение:** [ias_uch_vnii/](ias_uch_vnii/) — веб-приложение на PHP (Yii2), PostgreSQL.
- **Документация:** [docs/README.md](docs/README.md) — навигатор по всей документации.
